## 互联网（Internet）
network application
最常用的通信模型是双向可靠的字节流，即两台主机间在建立链接后双向的、可靠的对字节流的读写
eg1：万维网
其使用了http的超文本传输通信协议，内容均为ASCII文本。客户端发送一个对网页的请求get，请求其根页面，服务器接收到请求后会检测其是否有效，并返回响应，响应中包含一个数字代码，表示响应的结果
