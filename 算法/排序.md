# 排序
## 1.插入排序
### 1.1直接插入排序
依次将每个元素插入到前面有序的部分，时间复杂度O(n^2)，为稳定排序
```c++
template <typename T>
void insertionSort(vector<T>& arr) {
    int n = arr.size();
    for (int i = 1; i < n; i++) {
        T key = arr[i]; // 当前待插入元素
        int j = i - 1;  // 从当前元素的前一个开始比较
        
        // 将比key大的元素向后移动
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        
        // 插入元素到正确位置
        arr[j + 1] = key;
    }
}
```
